# 🎨 MonaLiza Filter Gallery

A creative image filter showcase built with HTML, CSS, and modern visual effects.  
This project demonstrates how different CSS filters can transform a single image into multiple artistic styles.

---

## 🔗 Live Demo
👉 https://behrokh-dev.github.io/MonaLiza/

---

## ✨ Features
- Multiple CSS filter effects applied to images  
- Smooth hover interactions and transitions  
- Pure HTML & CSS (no frameworks or libraries)  
- Responsive and lightweight design  
- Focus on visual experimentation and creativity  

---

## 🛠️ Built With
- HTML5  
- CSS3 (Filters, Transitions, Flexbox/Grid)

---

## 💡 What I Learned
- Advanced usage of CSS `filter` property  
- Creating smooth hover animations  
- Improving UI presentation with minimal and clean code  
- Experimenting with visual storytelling using CSS
